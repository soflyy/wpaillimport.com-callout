Callout block for WP All Import's documentation.

== Description ==

This is a custom Gutenberg block created for wpallimport.com's documentation.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/callout` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
